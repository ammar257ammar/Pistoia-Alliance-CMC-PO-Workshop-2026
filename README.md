## Overview
 
This repository contains all materials from a hands-on workshop introducing the **CMC Process Ontology (CMC-PO)**, developed by the Pistoia Alliance. The workshop guided participants through the annotation of a real pharmaceutical manufacturing process description using controlled ontology terms.
 
### The Annotation Excercise
 
Participants were then given two sheets:
 
1. **A process description** — a written description of a Paracetamol API synthesis procedure.
2. **A PSOA reference sheet** — a curated list of ontology terms from CMC-PO covering process stages, operations, and actions.

The task was to annotate the recipe by mapping each step to the most appropriate ontology term. Where no suitable term existed, participants were encouraged to flag the gap — turning the exercise into a direct, crowd-sourced validation of the ontology's coverage and completeness.
 
A **sample annotation** was provided as a reference CSV, and a **Jupyter Notebook** demonstrated the full pipeline: from annotated CSV → RDF knowledge graph (serialised as Turtle) → interactive HTML visualisation of the PSOA hierarchy.

## About CMC-PO
 
The **Chemistry, Manufacturing and Controls Process Ontology (CMC-PO)** is a Pistoia Alliance initiative to create a shared, standardised vocabulary for describing pharmaceutical manufacturing processes. It supports semantic interoperability across regulatory submissions, process development, and manufacturing data systems.
 
For more information, visit [pistoiaalliance.org](https://www.pistoiaalliance.org).
